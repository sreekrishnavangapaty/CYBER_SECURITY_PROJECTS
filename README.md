# 🔐 Cyber Security Mini Projects

This repository contains three simple cybersecurity projects developed using Python. These projects demonstrate basic security concepts such as password validation, encryption techniques, and phishing detection.

---

## 📌 Project 1: Password Strength Checker
This program checks how strong a password is based on:
- Length of the password
- Presence of uppercase and lowercase letters
- Numbers
- Special characters

### Output Levels:
- Weak ❌  
- Moderate ⚠  
- Strong ✅  

---

## 📌 Project 2: Caesar Cipher Tool
This project allows users to:
- Encrypt text using a shift value  
- Decrypt text back to original  

### Concept:
Each alphabet is shifted forward or backward based on the given number.

---

## 📌 Project 3: Phishing Detection System
This program identifies suspicious messages using:

### Detection Rules:
- Common phishing keywords (urgent, verify, login)
- Email presence (@ symbol)
- Links (http/https)
- Suspicious patterns

### Output:
- Safe Message ✅  
- Suspicious Message ❌
